有时候并不需要整个网页中的数据，只要其中某些地方有用的数据，所以就需要
数据提取技术。
1. 基于正则表达式的的数据提取技术
2. 基于BS4的数据提取技术
3. 基于XPath的数据提取技术

三种技术可以共同使用，具体应用取决于要使用的效果

### 正则表达式
爬取的网页本质上是一串很长的字符串，使用正则表达式进行匹配很合适。

优点：速度快、效率高、准确性高

缺点：新手上手难度高

**正则注意** 
> 所有的元字符默认只匹配一位字符
> 
> 比如 \d、\w、\s 等等

如果想要匹配多位字符，就要使用量词
> "*", "+", "?" 等等

> [...] 代表字符组，只匹配字符组中的字符
> 
> [^...]匹配除了字符组中的所有字符

贪婪匹配
```python
.*
```
惰性匹配
```python
.*?
# ?的作用是尽可能少的去匹配*，让*匹配的少一点
```

爬虫使用最多的就是惰性匹配
